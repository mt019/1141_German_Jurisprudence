# **ADHGB, HGB, BGB 與羅馬法：契約類型分類體系之法源與演化**

本節將結合歷史法系與現行法典，徹底解構以下問題：  
→ **ADHGB、BGB、HGB 中的契約類型分類，是從羅馬法時期延續而來的嗎？**  
答案是：**是的，這些分類制度在結構與法理上，深受羅馬法之影響與繼承。**

***

## **一、羅馬法下的契約制度原型：Typenlehre（類型教義）**

### **1. 契約有效性限於「法定類型」**

羅馬法下，並非所有合意（Konsens）都構成契約，而只有少數被明文承認的**法定契約類型（Vertragstypen）**具法律拘束力：

| 類型 | 拉丁文原稱 | 簡要說明 |
|------|-------------|-----------|
| 買賣契約 | *emptio venditio* | 交換標的與價格 |
| 借貸契約 | *mutuum* | 金錢或可替代物的無息借貸 |
| 承攬契約 | *locatio conductio operis* | 提供特定工作成果 |
| 租賃契約 | *locatio conductio rei* | 出租財物使用 |
| 勞務契約 | *locatio conductio operarum* | 提供勞務但無保證成果 |
| 委任契約 | *mandatum* | 無償辦理他人事務 |
| 贈與契約 | *donatio* | 單方無償給予 |

此體系形成所謂的 **Typenzwang（類型強制）** 與 **numerus clausus（封閉數量）** 原則，即：  
→ **契約只能存在於法律承認的框架中。**

***

## **二、ADHGB（1861）：商法上的類型繼承與轉化**

### **1. 歐陸法系首次「商業契約法典」**

ADHGB 是第一部全德適用的商法典，其契約體系延續羅馬法精神，並加入商業習慣法之動態內容。

| 核心貢獻 | 說明 |
|----------|------|
| **Vertragstypenkatalog** | 如 § 271 ADHGB 已開始編列典型契約，實質繼承羅馬法框架 |
| **相對／絕對商業契約劃分** | 借鏡羅馬法對「當事人身份」的區分，區分是否商人及交易本質 |
| **商業契約特殊條款發展** | 對買賣、委託、運輸等進行習慣法補充與制度化整理 |

### **2. 契約類型繼承羅馬法結構**

下表比較 ADHGB 類型與羅馬法原型：

| ADHGB 類型 | 對應羅馬法 | 是否制度化類型 | 說明 |
|-------------|-------------|----------------|------|
| Kaufvertrag（買賣） | *emptio venditio* | 是 | 價金交付與標的轉移 |
| Werkvertrag（承攬） | *locatio conductio operis* | 是 | 工作成果之完成 |
| Darlehen（借貸） | *mutuum* | 是 | 金錢或替代物借貸 |
| Auftrag（委任） | *mandatum* | 是 | 商人代表交易處理事務 |
| Spedition（貨運代理） | 無直接對應 | 新創 | 從商業習慣演化出來的新型契約 |

## **三、BGB（1900）：全面私法體系化與 Typenordnung 編碼化**

### **1. BGB 第二編（Schuldrecht）作為契約法的核心**

| BGB 範圍 | 條文 | 說明 |
|----------|------|------|
| **Allgemeines Schuldrecht** | §§ 241–432 | 羅馬法上的 obligatio 通則，處理債的成立與履行等共通原則 |
| **Besonderes Schuldrecht** | §§ 433–853 | 編碼化的 Vertragstypen，承繼羅馬法核心範型，提供預設法律內容 |

### **2. Vertragstypen 立法化**

BGB 將下列契約作為基本類型典範：

| BGB 契約 | 對應羅馬法 | 條文範圍 |
|-----------|-------------|------------|
| Kaufvertrag | *emptio venditio* | §§ 433–479 |
| Werkvertrag | *locatio conductio operis* | §§ 631–650v |
| Mietvertrag | *locatio conductio rei* | §§ 535–580a |
| Dienstvertrag | *locatio conductio operarum* | §§ 611–630 |
| Auftrag | *mandatum* | §§ 662–674 |
| Schenkung | *donatio* | §§ 516–534 |
| Darlehen | *mutuum* | §§ 488–515 |

→ **清晰可見，BGB 是對羅馬法典型契約的系統化與現代化延伸。**

***

## **四、HGB（1900）：商業契約特化與補充系統**

### **1. 以 ADHGB 為基礎進行系統擴展**

HGB 的契約規範聚焦於商人之間的交易，建立**特殊契約模型**與**特則優先適用原則**。

| HGB 契約 | 羅馬法對應 | 說明 |
|----------|--------------|------|
| Handelskauf | *emptio venditio* | 商人間買賣特別規定（如檢查義務、瑕疵通報期限） |
| Kommissionsgeschäft | *mandatum* 的擴張 | 商人受他人委託以自己名義交易 |
| Frachtgeschäft | 羅馬法無明文對應 | 新興商業運輸契約（源自習慣法） |
| Speditionsvertrag | 同上 | 專業物流契約發展而來 |
| Handelsvertretervertrag | 羅馬法中未制度化 | 商業代理制度現代化立法成果 |

***

## **五、法系總結：契約類型秩序的歷史延續性**

| 層次 | 羅馬法 | ADHGB | BGB | HGB |
|------|--------|--------|-----|-----|
| 類型法典化 | 有（少數類型） | 開始類型建模 | 完整法典化 | 商事補充典型化 |
| 類型強制（Typenzwang） | 是 | 是 | 基本承認但非絕對強制 | 商業契約中仍具實務意義 |
| 混合契約容許 | 否（原則不容） | 否（較保守） | 是（可按近似性補充適用） | 是 |
| 非典型契約（Innominatvertrag） | 否 | 否 | 是（實務發展允許） | 是 |

***

## **六、結論：從羅馬法到現代德國契約分類的歷史傳承與創新**

- **羅馬法提供範型與結構理性（Typenlehre）**
- **ADHGB 為商業習慣法典化起點**
- **BGB 以體系法形式將類型範式全面法典化**
- **HGB 保留商人實務特色並將其系統整理**

因此，**當代德國民商法中的契約類型分類，不僅是實用功能需求，也是對古典羅馬法理之現代延續與轉化。**
